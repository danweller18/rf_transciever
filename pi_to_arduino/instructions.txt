
Place the Sketches in your normal Arduino Sketches location. Install any
required libraries:

-Arduino rc_switch library
https://code.google.com/p/rc-switch/


RPi_utils
From your RPi, clone this archive:

git clone git://github.com/ninjablocks/433Utils.git

cd 433Kit/RPi_utils

(As per the original rc_switch distribution) Follow the instructions to
install the wiringpi library. After that you can compile the example
programs by executing `make`.
https://projects.drogon.net/raspberry-pi/wiringpi/download-and-install/
